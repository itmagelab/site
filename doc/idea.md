# Идея проекта: Сайт для личного бренда iTMageLAB

## Описание

Проект представляет собой **сайт-визитку** для меня. Сайт написан на Rust с использованием фреймворка Yew и компилируется в WebAssembly для работы в браузере.

## Основная цель

Создание современного продвинутого сайта для личного использования, на котором я могу разместить резюме, добавить контакты, описать мою работу и представить свою карьеру.

## Обо мне

Я Senior DevOps, SRE с большим опытом работы в IT-индустрии. Сайт будет являться единой точкой информации обо мне6 а так же демонстрацией моих способностей в области DevOps.

Мои основные специализации, это:

- Администрирование Linux и других ОС семейства UNIX
- Установка, настройка и поддержка сложных систем

## Структура сайта

Сайт состоит из нескольких страниц, с разными наборами блоков.
Первая страница, приветственная, будет состоять из 3 блоков:

### 1. Hero Section (Заголовок и призыв к действию)

**Назначение**: первое впечатление, привлечение внимания и побуждение к действию

**Содержание**:

- Заголовок: "iTMageLAB"
- Подпись: "Семенов Андрей Владимирович"

**Дизайн**:

- Полноэкранный блок
- Стиль должен напоминать консоль
- Центрированный текст

### 2. Обо мне (About Section)

**Назначение**: представление меня, создание доверия

**Содержание**:

- Моя фотография (профессиональное фото)
- Заголовок: "Обо мне"
- Текст-описание:
  - Опыт работы по годам
  - Философия работы

**Дизайн**:

- Двухколоночная раскладка: фото + текст
- Светлый фон для легкости восприятия
- Акцентные элементы (иконки, цитаты)

### 3. Портфолио и навыки (Portfolio Section)

**Назначение**: Детализированное описание моих работ и достижений

**Содержание**:

- Заголовок: "Портфолио"
- Таблица с работами:
  - Ссылки, если есть
  - Описание
- Список навыков с полосой прогресса

**Дизайн**:

- Легкочитаемые таблицы с рамками

### 4. Контакты (Contact Section)

**Назначение**: предоставление всей необходимой информации для связи

**Содержание**:

- Заголовок: "Контакты"
- Контактная информация:
  - **Телефон**: с прямой ссылкой для звонка
  - **Email**: для вопросов и предложений
- Социальные сети:
  - Telegram (основной канал)
  - VK (опционально)

**Дизайн**:

- Иконки для каждого способа связи
- Кликабельные ссылки для быстрой связи
- Тёмный или акцентный фон для выделения

### Footer (Подвал сайта)

**Содержание**:

- Копирайт "© 2024 Я Инна. Все права защищены"
- Краткая информация о салоне
- Дублирование ссылок на социальные сети
- Кнопка "Наверх" для быстрого возврата

**Дизайн**:

- Тёмный фон
- Минималистичный дизайн
- Компактное расположение информации

## Дизайн и цветовая палитра

### Цветовая схема

- **Основной фон страницы**: Мятно-зеленый (`bg-green-200`, `#BBF7D0`)
- **Заголовки**: Темно-серый (`text-gray-800`, `#1F2937`)
- **Подзаголовки и текст**: Средне-серый (`text-gray-700`, `#374151`)
- **Акцентный цвет (ошибки)**: Красный (`text-red-600`, `#DC2626`)

### Шрифты

- **Основной шрифт**: Sansation (Google Fonts)
- **Fallback**: sans-serif

### Стилистика

- Чистый минималистичный дизайн без блоков
- Текст на фоне без дополнительных обрамлений
- Хорошая контрастность для читаемости (темный текст на светлом мятном фоне)
- Центрированный контент

## Технический стек

### Frontend

- **Язык**: Rust (edition 2024)
- **Фреймворк**: Yew (CSR mode)
- **Компиляция**: WebAssembly (WASM)
- **CSS**: Tailwind CSS (через CDN)
- **Сборка**: Trunk

### Основные зависимости

- `yew` — UI фреймворк
- `wasm-bindgen` — связывание Rust и JavaScript
- `web-sys` — Web API bindings
- `serde` / `serde_yaml` — парсинг контента

### Инфраструктура

- **Хостинг**: статический хостинг (Cloudflare Pages, Netlify, Github Pages)
- **SEO**: meta-теги, Open Graph
- **Аналитика**: Яндекс.Метрика

## Архитектура

## Преимущества решения

- **Производительность**: WebAssembly обеспечивает быструю загрузку
- **SEO-оптимизация**: правильные мета-теги для локального поиска
- **Мобильная адаптивность**: полностью адаптивный дизайн
- **Поддерживаемость**: контент отделен от кода (YAML)
- **Современность**: использование Rust + WASM
- **Безопасность**: типобезопасность Rust
